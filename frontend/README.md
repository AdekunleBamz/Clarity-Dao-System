# Clarity DAO Frontend

React-based frontend for the Clarity DAO System with WalletConnect integration.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```
VITE_WALLETCONNECT_PROJECT_ID=your_project_id
```

3. Run development server:
```bash
npm run dev
```

## Features

- WalletConnect integration for Stacks wallets
- Governance dashboard with proposal voting
- Treasury management
- Token balance and voting power display
